The Real Time Bus Tracker

This program is called Real Time Bus Tracker. The tracking application will take the bus route for a provided transportation are and animate the stops with a map marker. As of now the grid locations of the bus stops are preloaded into the source code. The area used is for the Victor Valley Transit located in Southern California. The image demonstrates the actual animated map in the web application. The links will navigate to the program or the GitHub Repository for the program on an external page. Take a look and I hope you enjoy.

Eyes

This program is called Eyes. This project uses a JavaScript function to control images created by code to follow mouse movement. This is a basic illustration on how The Document Object Model can be manipulated with code and user input. In the future these functions will be used in a mini web game. The buttons navigate to the APP, the GitHub repository, and various other projects on this page. If viewing on a portable device the eyes should not move as they need to follow a mouse.

PacMen

This project is called PacMen. The application creates new images randomly and moves them at different access points randomly. This project will lead to a fun widget with different illustrations added to it. To access the Pac Application, press the button below. To add images, press the add button on screen. To move the images, press the start button. To move faster press it again. See how many Pac images can fit on a screen.

PacMouse

This project is called PacMouse. The application uses an illustrated mouse head and a moving ghost. The project demonstrates the use of Z axis and also movement of an illustrated image across the screen. Click on the mouse head and watch him start chomping. The more clicks the mouse gets the faster he moves. The code uses edging so when the mouse hits the edge the illustration will reverse the image
